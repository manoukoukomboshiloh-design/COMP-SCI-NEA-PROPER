# Shiloh NEA Login System

This is a standard client-server login system using Python sockets and SQLite database.

#Project Structure

- the client will contains the client application
- the server will contains the server application
- and the database Contains database setup and data

#SETUP

1. Run the database setup:
   python login/database/setup_db.py

2. Start the server:
   python login/server/server.py

3. In another terminal, run the client:
   python login/client/client.py

# Usage

The client will ask for a username and password. Use the information in the database e.g  username is Shiloh, the password is SHILH456!!. (aint a actual password g)
